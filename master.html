Exporter les données

Pour exporter les données, adaptez le modèle d'exportation
Chaque champ sera remplacé par les données correspondantes
Un modèle complet :
nom;prenom;classe;nais;sexe;temps;datas

N'utilisez les champs datas et temps que si ils sont présents dans les données.

Importer des noms \ classes

les bases de nom des participants sont importées :
  • Soit à partir des exportations de classe de Pronote
  • Soit à partir de fichier csv comportant au minimum
    les champs : nom, prénom, classe

n'oubliez pas le nom de la classe pour les fichiers Pronote
